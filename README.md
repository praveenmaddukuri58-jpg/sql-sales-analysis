# 📊 Sales Trend Analysis — SQL Project

## Overview
Analyzed 12 months of retail sales data to uncover revenue trends, top-performing products, 
and regional performance gaps. Findings presented as actionable business insights.

## Business Questions Answered
- Which months had the highest and lowest revenue?
- What are the top 5 best-selling products?
- Which regions are underperforming vs. target?
- What is the average order value by customer segment?

## Tools Used
- SQL (SQLite)
- DB Browser for SQLite (free tool)

## Key Findings
- Q4 drove 38% of annual revenue, with December as the peak month
- 3 products accounted for 52% of total units sold
- The Southeast region underperformed by 18% vs. the company average
- B2B customers had 2.4x higher average order value than retail customers

## Files
| File | Description |
|------|-------------|
| `sales_analysis.sql` | All queries — trends, rankings, breakdowns |
| `sample_data.sql` | Script to create and populate the sample database |

## How to Run
1. Download and install [DB Browser for SQLite](https://sqlitebrowser.org/)
2. Run `sample_data.sql` to create the database
3. Run `sales_analysis.sql` to see all analysis queries and results
